Coding   Resources

{   Learn   }

freeCodeCamp

Provides completely free, interactive web development tutorials. Learn to code alongside instruction and with error checking.

Code Academy

Very similar to freeCodeCamp. They cover more topics, however not all topics/features are free.

Bento

Tons of curated, free web development tutorials and articles covering many different topics. There's tracks to follow for both front-end and back-end development. Comes with recommendations on what to learn next based on what you already know.

{   References   }

Stack Overflow

If you run into a problem or have a question, someone on this site has already asked and answered it. Post or search for a question and get answers from other developers.

W3schools

Offers reference guides for HTML, CSS, JS, and more. If you forget a tag or property name, check here.

{   Text   Editors   }

Both of these editors are free to download and are customizable. Add user built packages to help you write cleaner, quicker code.

Sublime Text

Atom

Atom is my personal favorite. I used to use Sublime Text, but now that I use GitHub, Atom pairs perfectly because it's software made by GitHub.
